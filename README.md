### Instructions to Run
1. Clone repo
2. Navigate in terminal to /Assignment1 or the (pom.xml root)
3. Run a [mvn compile] or [mvn verify]
4. To run app [mvn exec:java "-Dexec.mainClass=Assignment1JamesTaylor.Assign1"]
